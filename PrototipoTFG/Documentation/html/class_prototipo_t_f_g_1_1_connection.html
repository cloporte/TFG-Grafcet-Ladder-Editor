Detailed Description

A Connection is a group of Connectors and InterNodes. These represent a connection between two Logic Elements by using multiple lines and intermediate points. The Start and End of the Connection are references to LogicElement. The logic implies that the Starting element is followed by the Ending Element, which means the Start considers the End as a Next element, whereas the End considers the Start as a Previous element.
