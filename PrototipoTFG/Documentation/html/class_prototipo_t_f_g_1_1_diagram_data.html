Detailed Description

DiagramData is a group of ObservableCollections of all the Elements a Diagram has : Nodes, Transitions, InterNodes, Inputs, Outputs, NotInputs, NotOuputs, Connectors and Connections. It is all the data needed to save or load a Diagram.
